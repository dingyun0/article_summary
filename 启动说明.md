# 文章抓取工具 - Web版使用说明

## 快速开始

### 1. 安装依赖

```bash
cd /Users/donson/Documents/projects/yuque_article_fetcher
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问页面

在浏览器中打开：http://localhost:5000

## 功能说明

### 主要功能
- ✅ 输入文章链接，自动抓取内容
- ✅ 支持微信公众号文章
- ✅ 实时预览文章标题和内容
- ✅ 自定义保存文件夹路径
- ✅ 保存为Markdown格式，包含原文链接
- ✅ 美观的用户界面

### 使用步骤

1. **输入链接**：在"文章链接"输入框中粘贴文章URL
2. **选择文件夹**：在"保存文件夹"输入框中输入保存路径（可选，默认为 `articles` 文件夹）
3. **抓取文章**：点击"抓取文章"按钮
4. **预览内容**：查看抓取到的文章标题和内容
5. **保存文件**：点击"保存到本地"按钮，文件将保存到指定文件夹

### 支持的链接类型

- 普通网站文章
- 微信公众号文章（https://mp.weixin.qq.com/）
- 博客文章
- 新闻文章

### 文件保存格式

保存的Markdown文件包含：
- 文章标题
- 原文链接
- 文章内容

## 注意事项

- 确保网络连接正常
- 某些网站可能有反爬虫措施，导致抓取失败
- 微信公众号文章需要能够在浏览器中正常访问
- 文件夹路径可以是相对路径或绝对路径

## 故障排除

### 抓取失败
- 检查链接是否正确
- 确认网站是否可以正常访问
- 尝试在浏览器中打开链接验证

### 保存失败
- 检查文件夹路径是否有写入权限
- 确认文件夹路径格式正确

### 服务无法启动
- 确认已安装所有依赖包
- 检查5000端口是否被占用
- 查看终端错误信息

## 技术栈

- 后端：Flask (Python)
- 前端：HTML + CSS + JavaScript
- 抓取：BeautifulSoup4
- 跨域：Flask-CORS
